# 🚀 DSALytics – DSA Performance Tracking Platform

DSALytics is a full-stack web application designed to track, analyze, and improve Data Structures & Algorithms (DSA) performance using LeetCode data.

## 🔥 Features

- 📊 Track LeetCode problem-solving stats
- 📈 Streak tracking system
- 🧠 Topic-wise performance analysis
- ⚔️ Friend challenge system
- 🎯 Placement-focused recommendations

## 🛠️ Tech Stack

- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: PostgreSQL (Neon DB)
- API Integration: LeetCode
